# CourseFlow

A calm, modern course planning experience designed to reduce stress and help students make better decisions.

---

## ✨ Overview

CourseFlow is a concept web app that rethinks how students plan courses and certifications.  
Instead of overwhelming users with dense schedules and cluttered interfaces, CourseFlow focuses on **clarity, calm design, and guided decision-making**.

This project explores how thoughtful UI/UX can turn a stressful process into a more manageable and even enjoyable experience.

---

## 🎯 Problem

Most course planning tools:
- feel cluttered and overwhelming
- rely on dense tables and rigid layouts
- increase cognitive load during already stressful decisions

Students often:
- struggle to visualize schedules
- miss conflicts
- feel rushed or uncertain when choosing courses

---

## 💡 Solution

CourseFlow introduces a more **calm and structured planning experience**:

- 🧠 **Guided flow** for choosing paths and building schedules  
- ⚡ **Smart recommendations** based on different lifestyles  
- 📅 **Conflict-aware planning** to reduce scheduling errors  
- 🎨 **Soft, minimal UI** designed to lower stress and improve clarity  

---

## 🎨 Design Philosophy

This project was built around one core idea:

> Planning your future should feel calm, not chaotic.

Key design decisions:
- **Soft color palette** (baby blue + mint green) to create a peaceful tone  
- **Generous spacing** to reduce cognitive overload  
- **Clear hierarchy** to guide attention naturally  
- **Minimal UI** to avoid unnecessary complexity  

---

## 🧩 Features

- Landing page with clear value proposition  
- Path selection flow  
- Course planner interface  
- AI-style recommendation results  
- Case study ("About") page explaining product thinking
